# 🐺 Lupus Analytic Agent & Market Analysis

## 📌 O mnie
Jestem pasjonatem analizy danych z ponad 20-letnim doświadczeniem w sektorze bankowym (DNB Bank). Łączę wiedzę z zakresu bezpieczeństwa fizycznego i operacyjnego z nowoczesnymi narzędziami programistycznymi (Python, SQL, Tableau).

## 🛠 Projekt: Lupus
Lupus to inteligentny agent monitorujący napisany w języku Python. 
- **Automatyzacja:** Skanowanie skrzynki odbiorczej (IMAP) pod kątem interesujących mnie wiadomości.
- **Bezpieczeństwo:** Implementacja zmiennych środowiskowych do ochrony danych wrażliwych.

## 📊 Analiza Rynku: AirBNB Nowy Jork
W ramach projektu analitycznego przeprowadziłem badanie rynku najmu krótkoterminowego na zbiorze ponad 100 tys. rekordów.

### Kluczowe wnioski inwestycyjne:
* **Budżet do 1 mln USD:** Rekomendowany kierunek to **Brooklyn** (wysoka stopa zwrotu ROI i duża dynamika ocen).
* **Budżet powyżej 1 mln USD:** Skupienie na **Manhattanie** w celu ochrony kapitału i maksymalizacji obłożenia całorocznego.
* **Ostrzeżenie:** Należy omijać rejony Staten Island ze względu na niską płynność turystyczną.
